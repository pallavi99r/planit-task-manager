# PlanIt 🗂️

PlanIt is a simple and clean task management web application that helps users organize daily tasks efficiently.

## 🚀 Features

- Add new tasks
- Edit tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML
- Bootstrap

## 📂 Project Structure

planit-task-manager
│
├── app.py
├── todo.db
│
├── templates
│   ├── index.html
│   └── edit.html
│
└── static
    └── logo.png

## ▶️ How to Run the Project

1. Clone the repository

git clone https://github.com/pallavi99r/planit-task-manager.git

2. Go to the project folder

cd planit-task-manager

3. Install dependencies

pip install flask

4. Run the application

python3 app.py

5. Open in browser

http://127.0.0.1:5001
